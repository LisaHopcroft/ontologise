About

Ontologise is a project developed to apply computational techniques to exploit the richness of historical data sources.

For more information, see:

  • Github repo

  • PyPI package

Command line arguments

Parsing Ontologise markup documents

usage: parse.py [-h] -i FILE [-s SETTINGS YAML] [-p] [-q] [-l LOGGING LEVEL]

Named Arguments

-i

A file using the Ontologise markup format

-s

The settings for the parsing

-p

Pause after every line

Default: False

-q

Suppress any logging messages

Default: False

-l

Possible choices: critical, error, warn, warning, info, debug

The logging level
